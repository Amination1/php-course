دوره کامل HTML

جلسه ۱: مبانی HTML و ساختار کلی صفحات وب

🎯 اهداف یادگیری:

  • درک مفهوم تگ‌های HTML
  • آشنایی با ساختار سلسله مراتبی تگ‌ها
  • ایجاد اولین صفحه وب

🔥 مفاهیم پایه:

هر سند HTML از تگ‌های تو در تو تشکیل شده است. ساختار اصلی شامل:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>عنوان صفحه</title>
</head>
<body>
    <!-- محتوای اصلی -->
</body>
</html>
📌 نکته حیاتی: همیشه تگ <!DOCTYPE html> را در خط اول قرار دهید

🧑‍💻 تمرین عملی:

صفحه‌ای ایجاد کنید که شامل:

  • عنوان صفحه به نام "صفحه تمرینی"
  • یک عنوان h1 با متن "سلام دنیا!"
  • یک پاراگراف با متن دلخواه
جلسه ۲: متون و قالب‌بندی پیشرفته

📚 تگ‌های متنی پیشرفته:

برای قالب‌بندی متن از تگ‌های زیر استفاده می‌کنیم:

<p>این یک متن <strong>پررنگ</strong> است</p>
<p>متن <em>مورب</em> و <u>زیرخط دار</u></p>
<blockquote>این یک نقل قول بلند است</blockquote>
<pre>متن پیش‌فرض با فونت مونواسپیس</pre>
📌 تفاوت strong و b: تگ strong برای اهمیت سمانتیک استفاده می‌شود

🧑‍💻 تمرین عملی:

صفحه‌ای ایجاد کنید شامل:

  • یک لیست شماره‌دار با ۳ آیتم
  • یک نقل قول با استفاده از blockquote
  • متن ترکیبی با strong و em
جلسه ۳: لینک‌ها و تصاویر HTML

🔗 لینک‌ها:

برای ایجاد لینک‌ها از تگ <a> استفاده می‌کنیم:

<a href="https://www.example.com">لینک به Example</a>

📷 تصاویر:

برای اضافه کردن تصاویر از تگ <img> استفاده می‌کنیم:

<img src="image.jpg" alt="تصویر مثال">

🧑‍💻 تمرین عملی:

صفحه‌ای ایجاد کنید شامل:

  • لینک به یک وب‌سایت خارجی
  • تصویر از اینترنت یا کامپیوتر خود
جلسه ۴: جداول و فرم‌های پیشرفته

📊 جداول:

برای ایجاد جداول از تگ‌های <table>, <tr>, <th>, و <td> استفاده می‌کنیم:

<table>
    <tr>
        <th>نام</th>
        <th>سن</th>
    </tr>
    <tr>
        <td>علی</td>
        <td>۲۵</td>
    </tr>
</table>

📝 فرم‌ها:

برای ایجاد فرم‌ها از تگ‌های <form>, <input>, <textarea> استفاده می‌کنیم:

<form>
    <input type="text" name="name" placeholder="نام">
    <textarea name="message">پیام خود را وارد کنید</textarea>
    <button type="submit">ارسال</button>
</form>

🧑‍💻 تمرین عملی:

صفحه‌ای ایجاد کنید شامل:

  • یک جدول با دو ستون (نام و سن)
  • یک فرم با فیلدهای نام، پیام و دکمه ارسال
جلسه ۵: عناصر چندرسانه‌ای

🎥 ویدیوها:

برای اضافه کردن ویدیو از تگ <video> استفاده می‌کنیم:

<video controls>
    <source src="video.mp4" type="video/mp4">
    <پشتیبانی نمی‌شود></video>

🎶 صدا:

برای اضافه کردن فایل‌های صوتی از تگ <audio> استفاده می‌کنیم:

<audio controls>
    <source src="audio.mp3" type="audio/mp3">
    <پشتیبانی نمی‌شود></audio>

🧑‍💻 تمرین عملی:

صفحه‌ای ایجاد کنید شامل:

  • یک ویدیو با دکمه کنترل
  • یک فایل صوتی قابل پخش
جلسه ۶: متون، لینک‌ها و تصاویر در HTML

🔗 لینک‌ها در تصاویر:

برای اضافه کردن لینک به یک تصویر از تگ <a> همراه با <img> استفاده می‌کنیم:

<a href="https://www.example.com">
    <img src="image.jpg" alt="تصویر">
</a>
جلسه ۷: متون، لینک‌ها و تصاویر در HTML

📄 فایل‌ها:

برای لینک دادن به یک فایل دانلودی از تگ <a> با ویژگی download استفاده می‌کنیم:

<a href="file.pdf" download>دانلود فایل PDF</a>
